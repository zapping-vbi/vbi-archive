Einige Fragen zu diesem Programm

Wofür ist diese Software?

Zapping ist ein TV Betrachter und somit ist die Hauptaufgabe von Zapping es zu ermöglichen, über ihre TV Karte am Computer fernzusehen. Nichtsdestotrotz, kann man mit einem Image Stream (Bilderstrom) eine Menge anstellen, Zapping bietet eine Möglichkeit diesen zu verarbeiten (fernsehen),man braucht jedoch nur ein Plug-in das ihren Wünschen entspricht und Sie können noch weitaus mehr mit Zapping machen. Sollte dieses Plugin welches Ihren Wünschen entspricht nicht existieren, sind Sie herzlich eingeladen eines zu schreiben.

Ich vermisse die Funktion XYZ, wird sie in Zukunft vorhanden sein?

Vielleicht, je eher Sie danach fragen, desto wahrscheinlicher ist es, das die Funktion in der nächsten Ausgabe enthalten sein wird

Das Programm funktioniert nicht!?

Bei mir läuft es ;-)
Wenn Sie einen Fehler (Bug), melden prüfen Sie, ob es sich nicht um einen schon bekannten Fehler handelt. (gucken Sie in der Datei BUGS nach, die bei Zapping mit dabei ist) und geben Sie soviele Informationen wie möglich an. Bei einem Absturz, geben Sie bitte an, was Sie genau gemacht haben, damit wir den Vorgang rekonstruieren können).
  • Wenn der Vollbildmodus nicht den ganzen Bildschirm füllt...
    Das kommt normalerweise dann vor, wenn Sie eine hohe Auflösung fahren. Schalten Sie den X Server in einen niedrigeren Auflösungsmodus um(unter /etc/X11/XF86Config, mit Xconfigurator oder einem ähnlichem Programm. Unter SuSe können Sie das auch mit Sax2 machen, indem Sie dort mehrere Auflösungen einstellen, Sie können dann unter X mit
    <CTRL>+ <ALT> + <+> oder <CTRL>+<ALT> + <-> in einen anderen Auflösungsmodus wechseln).
  • Das Vollbild und Vorschaubild sind stark gestört
    Dies tritt normalerweise auf, wenn Zapping nicht den richten BPP Modus finden kann, den der X Server intern benutzt. Dies kommt häufig im 24/32 bpp Modus vor, solange es keinen mir bewußten Weg gibt, dies zu umgehen, versuchen Sie den Kommandozeilen Aufruf --bpp zu übergeben.
  • Ich habe eine NTSC Kamera an S-Video angeschlossen, aber PAL ist belegt.
    Dies ist ein Fehler in der V4L Spezifikation. Sie können dies umgehen indem Sie die Angaben für tunerfreie Eingaben als Komandozeile übergeben. Zum Beispiel:
    zapping -n SECAM
    Bemerkung: Dies ist nur eine Umgehungsmöglichkeit, die nur funktioniert wenn Sie mindestens einen Eingang mit dem Tuner haben.

Können Sie mir eine kurze und prägnante Beschreibung von Zapping geben?

Zapping ist ein TV Betrachtungs Programm für Gnome. Es ist auch ein komfortables framework Programm um den image stream von der TV Quelle zu erhalten und zu verarbeiten (z.B es ist eine Webcam an Ihre V4L Schnittstelle angesclossen, oder teletext an den vtx Schnittstellen).
