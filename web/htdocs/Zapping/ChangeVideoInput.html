Change Video Input and Video Standard

You can select the current video input and video standard from the respective submenus of the Channel menu. Additionally video inputs can appear on the channel list. An entry for the S-Video input may look like this in the channel editor:

  Channel name Video input RF Channel Video standard Accelerator
0 svideo S-Video (not applicable) NTSC Alt+Shift+s

Now you can select the S-Video input by pressing Alt+S, 0 on the numeric keypad or the 0 button on your remote control. When your remote has an S-Video button you can also bind the commands zapping.set_channel(0) or zapping.lookup_channel('svideo') to it.

If changing channels does not work as expected please check if do not change has been selected as video input or video standard.
