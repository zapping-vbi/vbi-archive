Using Zapping with Nxtvepg

Since 2004-05-18 Zapping 0.7 can cooperate with the Nxtvepg application. You can run Nxtvepg and Zapping concurrently, switch channels from within Nxtvepg, and Zapping can display information about the program when you switch channels.

  • You will need Nxtvepg 2.7.1pre1 or later.
  • Only one application can open a VBI device. It is not possible to browse Teletext or see Caption and Subtitles while updating the Nxtvepg database. You can temporarily disable VBI access in the Zapping preferences. This problem is being worked on, see the ZVBI Proxy page.

To configure Nxtvepg for Zapping choose Configure >> TV app. interaction, and change TV application to Zapping.
