Requirements

  • To compile you will need GCC, Perl, the standard Unix tools, Doxygen if you want to generate documentation.
  • At runtime the library needs libpng and Posix threads (will be removed in the next major version).

The library has been tested on Linux and FreeBSD.

Binaries

If possible you should use binaries which have been created for your distribution. Since XawTV requires libzvbi nearly all distros should have it.

Source Code

You can download the latest version of libzvbi from our Sourceforge project page.

Anonymous CVS Access

To get the latest development version of libzvbi you can access the CVS server anonymously. If you are unfamiliar with using the CVS utility and the CVS commands go to the CVS web site for further information.

Step 1: Set the $CVSROOT environment variable. For sh/ksh/bash: 

export CVSROOT=:pserver:anonymous@cvs.sf.net:/cvsroot/zapping

For csh/tcsh: 

setenv CVSROOT :pserver:anonymous@cvs.sf.net:/cvsroot/zapping

If you don't want to set $CVSROOT you can use the cvs -d option: 

cvs -d :pserver:anonymous@cvs.sf.net:/cvsroot/zapping login etc

Step 2: To log in to the Zapping CVS server type the following. When asked for a password just press enter. 

cvs login

Step 3: To download the latest version of libzvbi type: 

cvs co vbi

You can request another version or development branch using the co -r TAG MODULE notation. This command for example requests libzvbi version 0.2.8: 

cvs co -r zvbi-0-2-8 vbi

To find out which tags are available you can use the cvs status command or browse the CVS repository. Select a module and open the tag menu at the bottom of the page.

From now on you can type

cvs update -d -P

in the respective source directory to update your copy of the CVS repository. As before, -r will choose a particular version or branch and -A the latest version.

Step 4: Prior to the first compilation run

./autogen.sh

to create missing configuration files. You will need several GNU build tools. The autogen.sh script tells you if any are missing or too old.
