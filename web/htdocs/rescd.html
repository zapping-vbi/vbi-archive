ResCD

I wrote ResCD (Resident CD player) as a small but powerful TSR for playing CD's in my old 486 using DOS. The code is written completely for TASM 5.0 using my beloved rhide and it works for MSDOS 6.2. It requires MSCDEX and HIMEM being installed.

Highlights:

  • It's a resident program, just load it in autoexec.bat and it will be present during the entire DOS session.
  • Very easy usage:
    • F12: Enters the text-mode interface with color menus, playlist, etc.
    • F11: Toggles secure mode.
    • F10: Stop/Resume
    • F9: Open/Close CD tray.
  • Should work under all programs that don't alter the interrupt table directly, but use the DOS ints instead. I.e., works under most programs.
  • Uninstall it running `rescd' again.
  • It's a good source for learning TASM (or 386 assembler in general)
Lowlights:
  • It has stayed unmaintained for more than two years now.
  • Some gross bugs:
    • It doesn't work under Win31
    • Hangs if you open the interface (F12) and you don't do anything in a while.
  • It's written in Spanish. Nevertheless, my brother has translated the GUI into English.
Anyway, I still use it regularly and find it very useful (and it was a great fun writing it :-).
