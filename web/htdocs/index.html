Zapping is a TV viewer for the Gnome environment. It has some interesting features, namely:

  • It is for Gnome / GTK. This gives the program a very nice look, as you can see in the screenshots page.
  • It is based on plugins. This will make it easy to add functionality to the program, without making it hard to maintain, such as saving video as AVI/MPEG, viewing mirrored TV (scanning from right to left) or whatever you can imagine (and program).

Requirements:
There are some requirements for compiling and running Zapping:

  • V4L or V4L2 support for your TV card.
  • Gnome libs >= 1.0.5
  • libglade >= 0.9
  • libxml (aka gnome-xml) >= 1.7.3
  • GdkPixbuf >= 0.8
  • libunicode >= 0.4
  • libjpeg and libpng are optional, but highly recommended.
These libraries are the standard GNOME ones. If you don't have them, an easy way to fetch them is to get a Ximian GNOME desktop, following this link.

Alternatively, if you have an RPM based distro, you can look for the packages using RPMFIND.

The remote control plugin needs the LIRC (Linux Infrared Remote Control) package. If not already part of your distribution you can download LIRC from http://www.lirc.org.

Downloading:
There are several ways to download zapping:

  • CVS: To get the latest development version of zapping, you can access anonymously the cvs server. Remember that there is no warranty that it will even compile. This method is mainly for developers and people that want to have the very latest, probably very buggy, version. Use the following:
    cvs -z3 -d:pserver:anonymous@cvs.zapping.sourceforge.net:/cvsroot/zapping login
    You will be asked for a password, just hit return.
    cvs -z3 -d:pserver:anonymous@cvs.zapping.sourceforge.net:/cvsroot/zapping co zapping
    This will checkout the latest zapping release, from now on you can use:
    cvs -z3 -d:pserver:anonymous@cvs.zapping.sourceforge.net:/cvsroot/zapping update -P
    to update your local copy of the repository.
    You can also browse the CVS repository, or get daily snapshots of the CVS tree.
